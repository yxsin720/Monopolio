## Descrição

Implementação do jogo Monopólio em C#.

O programa permite:

- Registo e listagem de jogadores
- Início de jogo com 2 a 4 jogadores
- Lançamento de dados e movimentação no tabuleiro
- Compra de espaços e casas
- Pagamento de aluguer
- Cartas de Chance e Community
- Aplicação das regras especiais (Start, Prison, FreePark, LuxTax)

## Estrutura do Projeto

O projeto segue a seguinte estrutura:

- src/ – código fonte
- src/Program.cs – ponto de entrada da aplicação
- controllers/ – lógica do jogo
- models/ – modelos de dados
- views/ – apresentação do tabuleiro

## Para executar o projeto

dotnet run --project Monopólio
